# Smart Expense Tracker

A full-stack expense management application built using React, Node.js, Express, MongoDB, and JWT Authentication.

## Features

- User Registration and Login
- JWT Authentication
- Dashboard Analytics
- Income Tracking
- Expense Tracking
- Budget Planning
- Interactive Charts
- MongoDB Database Integration
- Responsive UI

## Tech Stack

### Frontend
- React.js
- Axios
- React Router
- Recharts

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- bcrypt.js

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/smart-expense-tracker.git
```

### Install Frontend

```bash
cd client
npm install
```

### Install Backend

```bash
cd server
npm install
```

### Environment Variables

Create `.env` file inside server folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=mysecretkey
```

### Run Backend

```bash
cd server
npm run dev
```

### Run Frontend

```bash
cd client
npm run dev
```
